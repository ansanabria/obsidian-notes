# Codex CLI - Installation and Login

Back to [[Codex CLI Fundamentals]].

Install:

```bash
npm install -g @openai/codex
brew install --cask codex
```

Start:

```bash
codex
```

Useful account and maintenance commands:

```bash
codex login
codex logout
codex update
codex --version
```

For normal interactive use, ChatGPT sign-in is usually the simplest authentication path.
