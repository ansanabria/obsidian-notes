# Codex CLI Fundamentals

Codex CLI is OpenAI's terminal coding agent. It works inside a local workspace, reads project instructions, can inspect and edit files, and can run shell commands under sandbox and approval controls.

## Core Habit

Give Codex clear context, boundaries, and verification expectations. Let it inspect before editing, ask it to run relevant checks, then review the diff yourself.
