# Codex CLI - Main Commands

Back to [[Codex CLI Fundamentals]].

```bash
codex          # open interactive CLI
codex exec     # run non-interactively
codex review   # review changes
codex resume   # resume a previous session
codex fork     # fork a previous session
codex apply    # apply the latest Codex diff
codex login    # authenticate
codex logout   # remove credentials
codex update   # update Codex
codex mcp      # manage MCP servers
codex plugin   # manage plugins
codex completion # generate shell completions
```
